## Overview
This project integrates OpenWeatherMap API using Python and visualizes real-time weather data using Streamlit, Matplotlib, and Seaborn.

## Features
- Real-time weather data
- API integration
- Data visualization
- Streamlit dashboard
- CSV download feature

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib
- Seaborn
- OpenWeatherMap API
